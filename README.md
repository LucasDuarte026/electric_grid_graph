# BusNetworkSearch: AI Assignment 1 - Graph State Space Search

This project is part of a **Artificial Intelligence (IA) Assignment** focused on modeling a real-world problem as a state space and applying search strategies to find and analyze solutions.

## 📌 Project Goals
1.  **Modeling**: Represent the electrical power grid (IEEE 14-bus case) as a graph.
2.  **Search**: Implement one blind (uninformed) and one heuristic (informed) search strategy.
3.  **Implementation**: develop a functional and well-documented Jupyter Notebook (`BusNetworkSearch.ipynb`).
4.  **Analysis**: Compare performance and efficiency of the selected search algorithms.

## 📁 Project Structure
- `.gemini/`: Context and instructions for AI agents.
- `data/`: Raw electrical grid data (IEEE 14-bus, 300-bus).
- `docs/`: Reference documents and theoretical background.
- `BusNetworkSearch.ipynb`: The primary report and implementation.
- `README.md`: High-level project overview (this file).

## 🚀 Getting Started
1. Ensure the `.venv` is activated.
2. Install dependencies (NetworkX, Pandas, NumPy, etc.).
3. Open `BusNetworkSearch.ipynb` to view the implementation and analysis.
